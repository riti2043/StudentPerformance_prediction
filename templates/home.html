Student Performance Prediction

This application estimates a student’s academic performance based on key study-related inputs:

  • Number of hours studied
  • Previous test scores (in percentage)
  • Average sleep duration during exam season
  • Practice intensity through sample question papers

It was developed to demonstrate an end-to-end machine learning pipeline for student performance prediction using multivariate linear regression.

Predictions are generated and displayed alongside a comparative performance visualization.
